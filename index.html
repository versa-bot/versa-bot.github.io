Versa

Versa is a multifunction discord bot for (auto)moderation, security, utilities and fun.

Moderation

  • Clear This will delete the amount of messages.
  • Kick This will kick the user from the server.
  • Ban This will ban the user from the server.
  • Unban This will unban the user from the server.
  • Warn This will warn the user.
  • Add Role This will add the role mentioned to the user.
  • Remove Role This will remove the role mentioned from the user.

Fun

  • Gif Will show GIFs based on your search query.
  • Meme will show some memes.
  • Say The bot repeat the
  • Random Reddit Will show a random post from the subreddit you specified.
  • Destroy The bot will destroy itself.

Image

  • GIF Cat Some cat images.
  • GIF Dog Some dog images.
  • Meme Random post from r/memes.
  • Random Reddit Random post from a subreddit.

Bot

  • Invite This will send you a link with which you can invite Versa to your own server.
  • About It will show the info about the Versa.
  • Help Lists all commands.

Info

  • Avatar Gets covid-19 stats for your country.
  • Profile Gets info about the user.
  • ServerInfo Gets info about the server.
